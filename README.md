# DesignIndia ERP — Print & Packaging Management System

A comprehensive, enterprise-grade ERP (Enterprise Resource Planning) solution built for the print and packaging industry.

## Features

- **Dashboard** — Real-time KPI command center with revenue tracking, production pipeline, and alerts
- **Sales Order Management** — Create, track, and manage sales orders with document scanning
- **Artwork Tracking** — End-to-end artwork workflow from design to approval
- **Production Management** — Monitor production jobs, departments, and progress
- **Inventory Management** — Real-time stock levels, supplier invoice scanning, and reorder alerts
- **Client Management (CRM)** — Full client profiles with revenue analytics and order history
- **Billing & Invoicing** — Invoice generation, partial payment tracking, and printable A4 invoices
- **Reports** — Business analytics and performance insights

## Tech Stack

- React 19 + Vite
- Recharts (data visualization)
- Lucide Icons
- AES-256 Encrypted Local Storage

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
